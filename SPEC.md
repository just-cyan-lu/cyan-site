# cyan-site — 陆思源的个人网站

> 域名：`cyan.roco.fun`
> 技术栈：React + Vite + TypeScript
> 定位：个人博客 / 创意实验田

---

## 技术方案

| 技术 | 用途 |
|------|------|
| React 18 + TypeScript | 核心框架 |
| Vite | 构建工具 |
| React Router | 路由管理 |
| Framer Motion | 动画 |
| Emotion | CSS-in-JS 样式 |

---

## 页面

| 路径 | 说明 | 状态 |
|------|------|------|
| `/` | 主页（浮光 · 掠影） | ✅ |
| `/dream` | 三千凡梦（每周自动更新） | ✅ |

---

## 设计语言

**概念：浮光 · 掠影** — 轻盈、日常、有温度

### 配色（青橙）
| 用途 | 暗色 | 亮色 |
|------|------|------|
| 背景 | `#111118` | `#faf9f7` |
| 主强调（橙） | `#f07820` | `#e86c1a` |
| 次强调（蓝） | `#38b6ff` | `#1e88e5` |
| 正文 | `#e8e8f0` | `#1a1a2e` |
| 文字次级 | `#8888a0` | `#6b6b80` |

### 字体
- 标题：**Noto Serif SC**
- 正文：**Noto Sans SC**
- 英文/数字：**Space Grotesk**

### 主题
- 亮/暗色双主题，自动 8:00–20:00 亮色，其余暗色
- 支持手动切换（localStorage 记忆）
- 右上角太阳/月亮图标按钮

---

## 三千凡梦

每周日 10:10 (CST) 自动生成一期：
- **内容来源**：DREAMS.md 过去一周日记
- **文字**：提取诗意片段，重新组合成散文诗
- **图像**：用日记内容生成 AI 抽象图像
- **更新方式**：Cron 任务 → 生成 → SFTP 上传到服务器

数据文件：`/dream.json`
图像：`/assets/dream.jpg`
